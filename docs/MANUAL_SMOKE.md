# M1 手动冒烟测试

启动应用前确保 `poetry install` 已完成。

## 1. 启动 + 主题

- [ ] `poetry run comfy-mgr-gui` 启动 GUI
- [ ] 看到 NavigationDrawer（左边 3 项：环境管理、节点目录、设置）
- [ ] 默认 page = 环境管理
- [ ] 右上角语言切换为 English → 所有菜单/按钮变英文
- [ ] 切回 zh_CN
- [ ] 设置页 → 主题模式 → 选"深色" → 整个 UI 立即变深色
- [ ] 切回"浅色" → 立即变浅色
- [ ] 选"跟随系统" → 关 Windows 深色模式看效果

## 2. 创建环境

- [ ] 点 "+" 按钮 → CreateEnvDialog 弹出
- [ ] 填名字、Python 路径、ComfyUI 源、端口 → "创建"
- [ ] 环境出现在左侧列表
- [ ] 详情面板显示新环境信息

## 3. 启动 + 日志

- [ ] 选中一个环境 → 点"启动"按钮
- [ ] StatusIndicator 变绿，显示"运行中"
- [ ] 日志区开始显示实时日志
- [ ] 点"停止" → 状态变灰，显示"已停止"
- [ ] 日志停在停止时间点

## 4. 删除环境

- [ ] 选环境 → 点"删除"
- [ ] ConfirmDialog 弹出
- [ ] 确认 → 环境从列表消失

## 5. 节点目录

- [ ] 切到"节点目录"页
- [ ] 输入 `https://github.com/ltdrdata/ComfyUI-Impact-Pack` → 点"添加"
- [ ] 节点出现（git clone 会跑几秒）
- [ ] 点删除 → 节点消失

## 6. 设置

- [ ] 切到"设置"页
- [ ] 改数据库路径 → 点"迁移" → 成功提示
- [ ] 改主题/语言/日志级别 → 立即生效（部分需重启 GUI）

## 7. 错误展示

- [ ] 故意创建重名环境 → ErrorBanner 弹出，6 秒后自动消失
- [ ] 启动不存在的环境 → ErrorBanner 提示

## 8. GUI 重启恢复

- [ ] 启动 env1 → 关闭 GUI（不停止 env1）→ 重启 GUI
- [ ] env1 状态显示"运行中"
- [ ] 点"停止" → 收到 PROCESS_NOT_RUNNING 错误（因为 backend 没在内存中）
