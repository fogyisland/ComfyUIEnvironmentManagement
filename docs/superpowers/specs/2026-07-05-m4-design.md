## 0. 摘要

M4 把 UI 层从 PySide6+QML 整体迁移到 WPF + C#,**Python 后端(M0-M3 的所有 service/repo/DB/SQLite)完整保留**。WPF 通过本地 REST + WebSocket 与 Python service 通信。Service 作为 WPF 的**隐藏后台子进程**启动(CreateNoWindow=true),寿命与 WPF 绑定。

新增 / 改变:
1. Python 端:新增 `comfy_mgr.server` 包(FastAPI + uvicorn)
2. Python 端:bridge 层从 PySide6 `Signal` 解耦,改为 `bus.emit("ws.push", ...)` 单源
3. C# 端:全新 `.NET 8 self-contained WPF` 工程,MVVM 手写(无第三方 MVVM 框架)
4. i18n 统一:Qt Linguist `.ts` 单源,新增 `Tools.Ts2Resx` 生成 `.resx` 给 WPF
5. 激活 M3 deferred hooks:compat_api / catalog_cache_ttl / locked UI / folder_rename disable / Semver 比较 / AST_DYNAMIC 兜底
6. 清理 M3 review ledger 11 项 minor findings
7. Schema 升 v5(`version_history.pkg_version` + `scanned_nodes.disable_mode`)

### 2.1 关键决策

| 项 | 决定 |
|---|---|
| 启动模式 | WPF 启动时拉 Python service 作隐藏后台子进程 |
| 协议 | REST (POST/GET, JSON) + WS (events) |
| 进程寿命 | WPF 死了 Service 跟着停(WPF 关闭发 SIGTERM,3s 未退则 Kill) |
| 端口 | WPF↔Service 固定 127.0.0.1:7800 |
| 端口冲突 | 启动时检测占用 → 拒绝 + 提示 |
| 映射 | 1:1 mirror Bridge Slot → REST route |
| 状态推送 | 单条长连 WS `/ws/events`,事件通过 `channel` 字段路由 |
| i18n | .ts 单 source,脚本生成 .resx |
| 打包 | .NET 8 self-contained + zip 绿色版 |
| WPF 风格 | Material You 紫色(#6750A4),沿用 M1 决策 |

## 3. 进程模型与生命周期

### 3.1 启动序列

1. 用户双击 `ComfyUI Manager.exe` → .NET runtime 起来 → WPF 加载
2. `App.OnStartup` 触发:
   - 探测端口 7800 是否被占用 → 占用则抛 `PORT_IN_USE`,弹错误对话框,退出
   - `Process.Start("python\python.exe", "-m comfy_mgr.cli serve --port 7800 --bind 127.0.0.1")`,参数 `CreateNoWindow=true`,无控制台窗口
   - 设 `PYTHONPATH` = `<root>;<root>\src`
   - 工作目录 = 项目根
3. WPF 调 `GET http://127.0.0.1:7800/healthz` 轮询(每 500ms,10s 超时)
4. 收到 200 → WPF 继续;超时 → 弹错,清理子进程,退出
5. WPF 创建 `ApiClient` + `WsClient`,WsClient 连接 `ws://127.0.0.1:7800/ws/events`
6. WPF 显示 MainWindow

### 3.2 关闭序列

1. WPF `MainWindow.Closing` 事件触发
2. 调 `service.CloseMainWindow()`(对无窗口进程无效)
3. 调 `service.Kill(entireProcessTree=true)`(SIGTERM 等价)
4. 等 3s,若未退出 → 强制 Kill
5. 同时:遍历 env list,对 status="running" 的 env 调 `ctx.process.stop(env_id, timeout=3)` 强制停子 ComfyUI 进程
6. WPF 退出

### 3.3 崩溃恢复

**WPF 崩**(AppDomain.UnhandledException):
- 写 `logs/wpf-crash.log`(完整 traceback)
- 同步停 Python service 子进程
- `Environment.Exit(1)`

**Service 崩**(Python 进程死,WPF 还活着):
- WPF 检测:WS 心跳 60s 无 `_ping` → 视为断开
- WPF 调 `GET /healthz` 失败 → 弹"Python 服务已停止,是否重启?"
  - 重启:`PythonLauncher.LaunchAsync()` 重新拉
  - 退出:优雅关 WPF

**Service 启动失败**:
- WPF 启动时 /healthz 超时 → 弹错对话框,不让进 UI
- 残留进程占用 7800(WPF 没干净退出导致)→ 提示用户手动清理

## 4. 通信协议总览

### 4.1 REST

- 端点前缀:`/api/v1/{bridge}/{slot-kebab}`
- HTTP 方法:POST(默认)/ GET(只读且无 body)
- Content-Type:`application/json`
- 响应 envelope(始终 200,业务错在 body):
  ```json
  {"ok": true, "value": <任意>}
  ```
  ```json
  {"ok": false, "error": {"code": "ENV_NOT_FOUND", "message": "..."}}
  ```
- 例外(基础设施层用 HTTP status):
  - 请求体 parse 失败 → HTTP 400 + `{ok: false, error: {code: "BAD_PAYLOAD"}}`
  - 路由不存在 → HTTP 404 + `{ok: false, error: {code: "ROUTE_NOT_FOUND"}}`
  - FastAPI panic → HTTP 500 + `{ok: false, error: {code: "INTERNAL"}}`

### 4.2 WebSocket

- 端点:`ws://127.0.0.1:7800/ws/events`
- 单向 server→client(client 不在 WS 发消息,所有调用走 REST)
- 单条长连,WPF 启动后立即建
- 消息格式:
  ```json
  {"channel": "<camelCase Signal>", "args": [<JSON args>], "ts": "ISO timestamp"}
  ```
- 心跳:server 每 30s 推 `{"channel": "_ping", "args": []}`
- 重连:WPF 端指数退避(1s → 2s → 5s → 10s → 30s,封顶 30s)
- 断线补偿:重连后调 `settings/get-all` + `env/list` + 各 env `process/get-status` 重建本地状态
- 序列化失败:server 端 try/except,降级 push `{channel: "_internalError", args: ["SERIALIZE_FAILED", str(exc)]}`,**不**关闭 WS

### 7.3 Bridge 重构(关键决策:B 方案)

`NodeBridge` 等 8 个 bridge 去掉 `QObject` 基类、`Signal()` 改为 `bus.emit("ws.push", name, *args)`、`Slot` 装饰器改为普通 method。WPF 端无 QObject 依赖;server 进程不启 Qt。

```python
# bridge/node_bridge.py (改后片段)
class NodeBridge:                       # 不再继承 QObject
    def __init__(self, m0_service, bus, ...):
        self.m0_service = m0_service
        self.bus = bus
        self._busy = False
        bus.on("versionChanged", lambda env_id, package:
               self.bus.emit("ws.push", "versionChanged", env_id, package))
        # 同理映射所有 Signal

    def list_versions(self, env_id, package):
        return self._invoke(self.version.list_status, env_id, package)

    def upgrade_node(self, env_id, package, target=None):
        return self._invoke(self.version.upgrade, env_id, package, target=target)
```

WS 广播器订阅 bus 的 `ws.push` channel:

```python
# server/events.py
class WSBroadcaster:
    def __init__(self, bus: EventBus):
        bus.on("ws.push", self._on_push)
        self._clients: set[WebSocket] = set()

    async def _on_push(self, channel: str, *args):
        msg = {"channel": channel, "args": list(args),
               "ts": datetime.now().isoformat(timespec="seconds")}
        dead = set()
        for ws in self._clients:
            try: await ws.send_json(msg)
            except: dead.add(ws)
        self._clients -= dead
```

### 7.4 Route 写法

```python
# server/routes/node.py
router = APIRouter()
def node_bridge(request: Request) -> NodeBridge:
    return request.app.state.node_bridge
```

### 7.5 Lifespan

```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    ctx: AppContext = app.state.ctx
    recover_persisted_processes(ctx)
    app.state.ws_broadcaster = WSBroadcaster(ctx.bus)
    yield
    # graceful shutdown: 停所有 env 子进程
    for env in ctx.environment.list_all():
        if env.status == "running":
            try: ctx.process.stop(env.id, timeout=3)
            except: pass
```

### 8.3 PythonLauncher

```csharp
public class PythonLauncher : IDisposable
{
    private Process? _service;
    private readonly int _port;
    private readonly string _bind;

    public PythonLauncher(string projectRoot, int port = 7800, string bind = "127.0.0.1")
    {
        _port = port; _bind = bind;
        PythonExe = Path.Combine(projectRoot, "python", "python.exe");
        ProjectRoot = projectRoot;
    }

    public async Task LaunchAsync(CancellationToken ct = default)
    {
        if (IsPortInUse(_bind, _port))
            throw new ServiceLaunchException(
                $"端口 {_port} 已被占用,可能已有 ComfyUI Manager 实例在运行");

        var psi = new ProcessStartInfo
        {
            FileName = PythonExe,
            ArgumentList = { "-m", "comfy_mgr.cli", "serve",
                             "--port", _port.ToString(),
                             "--bind", _bind },
            WorkingDirectory = ProjectRoot,
            UseShellExecute = false,
            CreateNoWindow = true,
        };
        psi.EnvironmentVariables["PYTHONPATH"] =
            $"{ProjectRoot};{Path.Combine(ProjectRoot, "src")}";

        _service = Process.Start(psi)
            ?? throw new ServiceLaunchException("无法启动 Python 子进程");

        await WaitForHealthAsync(ct);
    }

    private async Task WaitForHealthAsync(CancellationToken ct)
    {
        using var http = new HttpClient { Timeout = TimeSpan.FromSeconds(2) };
        var deadline = DateTime.UtcNow.AddSeconds(10);
        while (DateTime.UtcNow < deadline)
        {
            if (ct.IsCancellationRequested) break;
            try
            {
                var r = await http.GetAsync(
                    $"http://{_bind}:{_port}/healthz", ct);
                if (r.IsSuccessStatusCode) return;
            }
            catch { }
            await Task.Delay(500, ct);
        }
        throw new ServiceLaunchException(
            $"Python service {_port} 端口 /healthz 超时(10s)");
    }

    public async Task ShutdownAsync()
    {
        if (_service is null or { HasExited: true }) return;
        try
        {
            if (!_service.WaitForExit(3000))
                _service.Kill(entireProcessTree: true);
        }
        catch { }
    }

    public void Dispose() => ShutdownAsync().GetAwaiter().GetResult();
}
```

### 8.5 ApiClient + WsClient

**ApiClient**:
```csharp
public class ApiClient
{
    private readonly HttpClient _http = new() {
        Timeout = TimeSpan.FromSeconds(30),
        BaseAddress = new Uri($"http://127.0.0.1:{port}/api/v1/"),
    };
    private static readonly JsonSerializerOptions Opts = new() {
        PropertyNamingPolicy = JsonNamingPolicy.CamelCase,
        Converters = { new JsonStringEnumConverter() }
    };

    public async Task<ErrorEnvelope> PostAsync<TReq>(
        string route, TReq body, CancellationToken ct = default)
    {
        var resp = await _http.PostAsJsonAsync(route, body, Opts, ct);
        var envelope = await resp.Content.ReadFromJsonAsync<ErrorEnvelope>(Opts, ct);
        return envelope ?? new ErrorEnvelope { Ok = false,
            Error = new ErrorBody { Code = "INTERNAL", Message = "空响应" }};
    }
}
```

**WsClient**:ClientWebSocket + handler dict + 指数退避重连 + 后台 ReceiveLoop。

## 11. Error Handling

### 11.1 ErrorCode 分类

| 类别 | 前缀 | HTTP status | WPF severity |
|---|---|---|---|
| 客户端输入 | `BAD_*` | 400 | info |
| 实体不存在 | `NOT_FOUND` | 404 | warn |
| 业务冲突 | `BIZ_*` | 409 | warn |
| 业务前置条件 | `PRECOND_*` | 422 | error |
| 外部 IO | `IO_*` | 500 | error |
| Git 子进程 | `GIT_*` | 500 | error |
| HTTP 外部 | `HTTP_*` | 502 | warn |
| DB 错误 | `DB_*` | 500 | error |
| 进程错误 | `PROC_*` | 500 | error |
| 服务内部 | `INTERNAL` | 500 | critical |
| WS 序列化 | `WS_*` | n/a | 静默 |

### 11.2 调用级 vs 全局错误

- **调用级**:`ApiClient.PostAsync` 始终返回 `ErrorEnvelope`,不抛异常;ViewModel 拿到 `{ok: False, error: ...}` 后 inline 处理(不通过 ErrorBanner 显示)
- **全局错误**:`bus.emit("errorOccurred", code, message)` 由 `_invoke` 自动 emit;`WSBroadcaster` 转发为 WS event;WPF `ErrorBannerViewModel` 收,按 severity 路由

### 11.3 ErrorCode 常量 — Single Source

放 `shared/errors.json`,Python + C# 都读:
```json
{
  "ENV_NOT_FOUND": {
    "http_status": 404,
    "severity": "warn",
    "i18n_zh_CN": "环境 {env_id} 不存在",
    "i18n_en_US": "Environment {env_id} not found"
  }
}
```

### 11.4 服务端日志

`logs/server-error.log`(RotatingFileHandler,10MB x 5)。触发级别:DB_* / INTERNAL / WS 序列化失败 / 进程清理失败。

### 11.5 崩溃恢复

WPF 崩 → 写 `logs/wpf-crash.log` + 停 service 子进程。Service 崩 → WPF WS 心跳超时 → 弹"重启/退出"对话框。
